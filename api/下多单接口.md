
## 下多单接口
POST
`
https://watbtc.vip/api/v1/orders/multi2
`

```
注：本接口用于下成交单，单次请求仅处理前十笔
```


```
  access_key: 从平台获取的access_key
  market: 市场，例如: btceth
  signature: 签名
  tonce: 时间戳，精确到秒
  orders: 多单数组的json字符串
```

orders参数示例

```json
[
	{
  		price: 价格,
	  	volume: 数量,
  		side: "buy"
	},
	{
	  	price: 价格,
  		volume: 数量,
  		side: "sell"
	}
]
```
