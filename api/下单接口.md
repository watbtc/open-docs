
## 下单接口
POST
`
https://rfinex.vip/api/v2/orders
`



```
  access_key: 从平台获取的access_key
  price: 价格
  volume: 数量
  market: 市场，例如: btceth
  side: "buy|sell"
  signature: 签名
  tonce: 时间戳，精确到秒
```
