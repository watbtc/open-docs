## 订单查询接口
GET
`
https://watbtc.vip/api/v1/orders
`

参数在query部分传递

```
  access_key: 从平台获取的access_key
  signature: 签名
  tonce: 时间戳，精确到秒
  
  state 状态，默认wait。支持 done/cancel
  market 市场，
  page 当前页，默认1
  limit 每页多少条记录，默认30
  order 排序方式，asc/desc二选一, 默认不传为 desc

## 根据最小订单号查询接口
GET
`
https://watbtc.vip/api/v1/orders/by_min_id
`

参数在query部分传递

```
  access_key: 从平台获取的access_key
  signature: 签名
  tonce: 时间戳，精确到秒
  
  state 状态，默认wait。支持 done/cancel
  market 市场，
  min_id 最小订单号，默认1
