
## 撤单接口
POST
`
https://rfinex.vip/api/v2/order/delete
`

参数在query部分传递

```
  access_key: 从平台获取的access_key
  signature: 签名
  id: 订单号
  tonce: 时间戳，精确到秒
```

```
        "code": "1000",
        "msg": "Success."
```

POST
`
https://rfinex.vip/api/v2/orders/delete
`


```
  access_key: 从平台获取的access_key
  signature: 签名
  ids: 订单号列表，以“,”分割，例子 123,456,789
```
